# Almotaliq Landing (React + Vite)

This is a responsive single-page landing site built with React, Vite, TailwindCSS and Framer Motion.
It is RTL-ready and includes sections for services, gallery, about, contact and more.

## Quick start

1. Install dependencies

```
npm install
```

2. Start development server

```
npm run dev
```

3. Build for production

```
npm run build
```

4. Deploy to Vercel: connect repository & Vercel will auto-detect Vite projects.

## Change contact info
Edit links in the components that reference phone/whatsapp and update as needed.
